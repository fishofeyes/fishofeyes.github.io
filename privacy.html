2. 我们收集的信息

我们郑重承诺，本扩展不收集、不存储、不以任何形式向我们的服务器或任何第三方服务器传输您的任何个人身份信息或浏览数据

具体说明：

  • 无数据收集：本扩展的程序代码在设计上即运行于您的浏览器本地。所有为实现功能而处理的数据（如有）均仅在您的设备本地进行处理，绝不会被发送到外部。
  • 权限使用说明：本扩展在Chrome Web Store中声明的权限（例如storage、activeTab等）仅用于实现核心功能。例如，storage权限可能用于在您的浏览器本地保存您的个人设置，这些数据永远不会离开您的计算机。

3. 信息的存储与保护

由于我们不收集您的任何数据，因此不存在数据传输或服务器存储的情况。

所有可能产生的数据（如本地设置）均加密存储在您的浏览器本地环境中，受您设备安全系统的保护。这些数据仅您本人可以访问，我们无法获取或查看这些信息。

4. 信息的使用与共享

我们不会使用或共享您的任何个人信息，因为我们根本不收集这些信息。

具体承诺：

  • 不收集个人信息或浏览数据
  • 不将数据发送到任何服务器
  • 不与任何第三方共享数据
  • 不进行用户追踪或分析

根据《通用数据保护条例(GDPR)》等法律法规，您享有以下权利：

  • 知情权：您有权了解我们如何处理您的数据（如上所述，我们不处理任何数据）
  • 访问权：您有权访问我们可能持有的您的任何数据（我们不持有任何数据）
  • 删除权：您可以随时通过浏览器设置管理或删除本扩展在本地存储的数据

6. 政策变更

如果本隐私政策发生更新，我们会在扩展的更新说明中提供提示，并更新在Chrome Web Store中的政策文本和生效日期。

我们会确保所有变更都符合Chrome Web Store的政策要求，并保持对用户数据保护的持续承诺。
